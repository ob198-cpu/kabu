今回の結論

半導体テーマは強いが、即時に購入候補へ昇格できる状態ではありません。

1年騰落率だけなら主比較+1%線を上回る銘柄が多い一方、SOX/NASDAQ/日経平均の下落日に強かったとは言えません。さらにレーザーテック、アドバンテストは割高・過熱・決算後反応のゲートで止まります。

次に進める価値があるのは、データが揃っているTDKの継続確認と、構造本命である東京エレクトロン・SCREENの決算/割高データ補完です。

接続フロー

1. 構造優位高シェア、不可欠工程、需要継続性を確認。ここでは加点しない。
2. 下落耐性SOX/NASDAQ/日経平均の下落日に、相対的に耐えたかを見る。
3. 決算・割高PER/PBR/ROE、売上・利益成長、決算後反応で止める。
4. +1%比較S&P500/日経平均の強い方を1%pt上回ったかを見る。
5. 6月再判定CPI、FOMC、日銀、日経75日線を見て、候補へ残すか決める。

数式・ルール

項目計算式扱い
構造優位構造スコア = 技術/シェア/不可欠性/需要継続/収益接続を0〜100点化単独では加点しない。候補追加の入口にだけ使う。
下落耐性下落耐性 = 指数下落日の1/5/20営業日超過リターン + 5日勝率 + 最大ドローダウン65点以上で通過、45〜64点は観察、44点以下は保留または停止。
+1%比較主比較+1%差 = 銘柄1年騰落率 - max(S&P500, 日経平均) - 1%pt通過しても購入候補確定ではない。割高・決算・下落耐性で止める。
半導体テーマ比較参考テーマ差 = 銘柄1年騰落率 - SMH騰落率 - 1%pt参考値。半導体テーマの中でわざわざ個別株を選ぶ根拠があるかを見る。
割高ゲートPER/PBR/ROEを確認。PER45倍以上またはPBR10倍以上は割高強め。割高強め、決算反応弱い、過熱ゲートありは候補昇格しない。
統合判断構造優位 → 下落耐性 → 決算/割高 → +1%比較 → 6月イベント後の最終確認足し算で一発採点しない。各ゲートを順番に通過したものだけ次工程へ進める。
