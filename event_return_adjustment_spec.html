何をする機能か

日銀・FRB・TDnetなどのイベント日と、候補銘柄の株価時系列を結合し、イベント後20営業日のリターンが日経平均を上回ったかを確認します。その結果を、6月テスト候補ランキングへ小さな補正点として反映します。

処理の流れ

1. イベント取得日銀・FRB・TDnet等の日付を記録
2. 株価取得候補銘柄と日経平均の1年日次株価を取得
3. リターン計算1営業日・5営業日・20営業日後を計算
4. 指数比較日経平均との差を超過リターンとして記録
5. 小さく補正最大±2.5点だけランキングへ反映

補正式

イベント反応補正 = 20営業日平均超過リターン補正 + 強弱回数補正
最終6月テスト点 = 基礎スコア + 本文イベント補正 + イベント反応補正
条件補正
イベント後20営業日の平均超過リターンが +5%以上+2.0点
イベント後20営業日の平均超過リターンが +2%以上 +5%未満+1.0点
イベント後20営業日の平均超過リターンが -2%以下 -5%超-1.0点
イベント後20営業日の平均超過リターンが -5%以下-2.0点
強いイベント回数が弱いイベント回数より2回以上多い+0.5点
弱いイベント回数が強いイベント回数より2回以上多い-0.5点
最終補正幅最大 +2.5点 / 最小 -2.5点

制限

この機能は因果関係の証明ではありません。イベント後に株価が動いた事実を記録するもので、決算内容、バリュエーション、為替、金利、指数全体の地合いと組み合わせて判断します。未来イベントや20営業日未満のイベントは未確定として扱います。
