投資分析モデル 学術的妥当性プロトコル

雰囲気のあるスコアから脱却し、再現性・検証可能性・反証可能性を持つ研究設計へ引き上げるためのプロトコルです。検証前の式は購入判断ではなく研究仮説として扱います。

基本方針

仮説を作る
→ 事前に評価式と除外条件を固定する
→ point-in-timeデータで検証する
→ 過去データでベンチマークに勝ったか確認する
→ 未知期間で外部検証する
→ 予測と実績の誤差を記録する
→ 採用/不採用を決める
検証前の式は、購入判断ではなく研究仮説です。

研究としての問い

問い検証方法
質的イベントは株価に影響するかイベントスタディで異常リターンを測る
その影響は一時的か、1年保有に使えるか1日、5日、20日、60日、250営業日で比較
量的スコアは将来リターンを説明するかランキング別ポートフォリオで検証
質的データは量的データに上乗せ価値を持つか量的モデル単独 vs 量的+質的モデルで比較
S&P500、日経平均、TOPIXを+1%以上上回れるか円建て・配当込み・コスト込みで比較

学術的方法

方法用途根拠
イベントスタディTOB、上方修正、新商品、戦争、金利イベント後の異常リターン測定MacKinlay 1997, Brown & Warner 1985
マーケットモデル個別株の通常リターンを市場リターンで説明し、異常リターンを出すEvent Study標準手法
Fama-French/Carhart補正サイズ、バリュー、モメンタムなど既知ファクターを控除Fama & French 1993, Carhart 1997
モメンタム検証過去リターンの継続性を検証Jegadeesh & Titman 1993
多重検定補正たくさん試した結果、偶然当たる問題を抑えるHarvey, Liu & Zhu 2016
ウォークフォワード検証過去で作った式を未来期間で検証実運用に近い検証

イベントスタディ

AR_i,t = R_i,t - E(R_i,t)

E(R_i,t) = α_i + β_i R_market,t

または

E(R_i,t) = α_i + β_m MKT_t + β_s SMB_t + β_h HML_t + β_mom MOM_t

CAR_i,[a,b] = Σ AR_i,t
期間意味
[0,1]即時反応
[0,5]短期反応
[0,20]約1か月の反応
[0,60]四半期程度の反応
[0,250]NISA 1年保有に近い反応

質的データの扱い

質的データは直接加点しません。
層役割スコアへの扱い
仮説データ層企業を探す入口原則、直接加点しない
実績データ層過去に効いたか検証質的評価の上限を決める
量的データ層最終判断の主軸購入候補スコアの中心
 
仮説S × 実績C = 質的Sにしない
仮説C × 実績S = まず再調査
実績なし = 購入候補スコアへ反映しない

モデル比較

モデル内容目的
BenchmarkTOPIX、日経平均、S&P500円建て何もしない場合との比較
Quant Only量的データだけ基礎モデル
Event Evidence Only実績イベントだけイベントの有効性確認
Quant + Event Evidence量的 + 実績イベント質的データが上乗せ価値を持つか確認
Quant + Hypothesis量的 + 仮説原則採用しない。危険性確認用
採用候補にできるのは、Quant + Event Evidence が Quant Only を未知期間で上回った場合だけです。

採用基準

項目採用基準
期待リターンベンチマークを年率または1年で+1%以上上回る
勝率ランダム選定や同業平均より有意に高い
最大下落ベンチマークより極端に悪くない
シャープ比ベンチマーク以上
多重検定補正後も有意性が残る
取引可能性売買代金、スプレッド、単元、NISA制約を満たす
再現性学習期間だけでなく検証期間でも機能する
