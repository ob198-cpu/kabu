4. 計算方法

探索スコア

探索スコアは、候補を広く見るための参考点です。未取得がある銘柄も、購入候補ではなく探索対象として残します。

NISA 1年保有スコア = 業績/質35% + 下落耐性25% + 割安20% + 中期トレンド10% + 決算後反応10%

購入候補スコア

購入候補スコアは、必須データがそろい、ハード除外に該当しない銘柄だけに出します。

購入候補スコア = 業績/質40% + 下落耐性30% + 割安20% + 決算後反応10%
停止条件 理由
必須データ未取得空欄を推測で補完しないため、購入候補スコアを出さない。
信頼度70点未満根拠の薄い数字を購入候補点に混ぜない。
過熱しすぎ1年上昇率が高すぎる銘柄は高値掴みリスクを確認する。
決算後反応が弱い市場が決算を評価していない可能性がある。
急落・割高リスク60日最大下落率、PER/PBR極端値を確認する。
