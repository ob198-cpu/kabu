この運用の合格ライン

「ただの投信運用」を上回れないなら、日本個別株を選ぶ意味が弱くなります。

比較対象目標判定
S&P500投信3か月で+1%ポイント以上上回る円換算で比較。負けるなら個別株比率を落とす。
日経平均連動投信3か月で+1%ポイント以上上回る日本株を選ぶ最低条件。
TOPIX連動投信3か月で+1%ポイント以上上回る大型株全体との比較。
合格 3か月後に、S&P500投信・日経平均/TOPIXのうち主要比較対象を+1%ポイント以上上回る。
不合格 4週間で指数に-3%ポイント以上負ける、または3か月で+1%ポイント超過できない。銘柄を削り、指数投信へ戻す。

全体の安全ルール

大きく負けないための停止条件です。

起きたこと行動理由
ポートフォリオ-5%新規買い停止。現金から最大3万円だけ追加可。普通の押し目か、失敗かを見分ける。
ポートフォリオ-10%半導体比率を下げる。追加買い停止。高PER株が崩れた時は守る。
日経平均が高値から-10%予定買いを半分にする。市場全体が悪い時は急がない。
日経平均が75日線を-2%以上下回る個別株の新規買い停止。指数トレンドが悪い時は勝率が下がる。

PERの見方
PERは「株価が利益の何年分まで買われているか」を見る指標です。PER20倍なら、現在の利益水準が続く前提で約20年分の利益を先に買っているイメージです。高PERそのものが悪いのではなく、利益成長率とセットで見ます。

1. 高PERが許される条件
売上成長率、営業利益成長率、EPS成長率が高く、次の2から3年も伸びる見通しがある時は、高PERでも正当化されます。例えばPER40倍でも、EPSが年20%以上伸びるなら市場は成長を買います。逆にEPS成長が1桁へ落ちると、PER40倍は割高に見えやすくなります。

2. 危険なのはPER上昇と利益鈍化が同時に起きる時
株価だけが上がってPERが高くなり、会社の利益見通しが伸びていない場合は危険です。市場が期待だけで買っている状態なので、決算で売上や営業利益が少し下振れするだけで株価が大きく下がります。

3. 金利上昇時はPERの許容水準が下がる
米10年金利が上がると、将来利益の価値が割り引かれます。特にPER30倍以上の成長株は、金利上昇だけで売られることがあります。利益が悪くなくても、金利だけで株価が下がる点に注意します。

4. 同業他社比較が必要
PERは単独では判断できません。同業平均より高いなら、その会社だけ利益率、成長率、ブランド、技術、還元力が本当に上かを確認します。味の素や任天堂のように期待が乗りやすい銘柄は、同業や過去平均との比較が必須です。

見る数字
予想PER、実績PER、EPS成長率、営業利益率、会社計画進捗率、同業平均PER、米10年金利を見ます。

この資料での対応ルール
PER30倍以上の銘柄は、営業利益見通しが前年比+10%以上、または会社計画進捗が順調な場合だけ追加候補にします。PER40倍以上で営業利益見通しが下方修正、または決算後に株価が-8%以上下落した場合は追加停止。米10年金利が4.70%以上なら、高PER銘柄の追加額を半分にします。 銘柄へ戻る
医薬リスク
医薬品株は景気に左右されにくく、守りと成長を両方持ちやすい分野です。一方で、普通の製造業とは違い、薬価、臨床試験、承認、特許、主力品依存で急に評価が変わります。中外製薬のような高品質株でも、医薬特有の事故リスクはあります。

1. 薬価改定リスク
日本では薬価改定により、販売数量が伸びても単価が下がることがあります。売上が横ばいでも利益率が下がる場合があり、医薬品株の見通しを読む時は「数量」と「単価」を分けて見ます。

2. 臨床試験リスク
新薬候補は、試験結果が良ければ大きな上振れ材料になりますが、失敗すると将来利益の期待が一気に消えます。第3相試験や主要評価項目の未達は、株価に大きなマイナスになりやすいです。

3. 承認・規制リスク
FDA、EMA、PMDAなどの承認が想定より遅れると、売上開始時期が後ろ倒しになります。承認されても適応範囲が狭い、警告表示が重い、競合薬に劣るなどの場合は期待が剥がれます。

4. 主力品依存と特許切れ
医薬品会社は、数個の大型薬が利益の大部分を支えることがあります。主力品の競争激化、特許切れ、後発薬やバイオシミラーの登場で、利益が落ちるリスクがあります。

5. 為替と海外売上
海外売上が大きい会社は、円高で円換算売上が減ります。円安は追い風ですが、為替で利益が増えただけなのか、本業の数量・単価が伸びたのかを分けて見ます。

見る数字
売上成長率、営業利益率、研究開発費率、主力品売上構成比、パイプライン進捗、薬価改定影響、為替感応度を見ます。

この資料での対応ルール
中外製薬は、営業利益見通しが前年比プラス、主力薬の売上が計画線、パイプラインで大きな失敗がない場合に追加候補とします。臨床試験失敗、承認遅延、営業利益見通し-10%以上悪化、または決算翌日に-8%以上下落した場合は追加停止。医薬イベント前は一括で買わず、予定額の半分までにします。 銘柄へ戻る
金利テーマ
銀行株は金利上昇で貸出金利と預金金利の差、つまり利ざやが広がりやすくなります。三菱UFJのようなメガバンクは、日銀の利上げや長期金利上昇で利益期待が高まりやすい一方、金利上昇が強すぎると逆に信用コストや債券評価損が問題になります。

1. 良い金利上昇
景気が崩れず、日銀がゆっくり正常化し、貸出需要が維持される金利上昇は銀行に追い風です。国内貸出利ざやが改善し、預金基盤の大きい銀行ほど利益が増えやすくなります。

2. 悪い金利上昇
金利が急に上がると、企業や個人の返済負担が増え、貸倒れリスクが上がります。不動産、建設、中小企業向け融資で信用コストが増えると、利ざや改善を打ち消します。

3. 債券評価損リスク
銀行は国債などの債券を多く持っています。金利が上がると債券価格は下がるため、保有債券の評価損が増えることがあります。実現損ではなくても、投資家は自己資本や含み損を警戒します。

4. 日銀方針の読み違い
市場が利上げを期待して銀行株を買っている時に、日銀が慎重姿勢を出すと失望売りが出ます。逆に急な利上げや円高ショックも、日本株全体にはマイナスになり、銀行株だけが上がるとは限りません。

見る数字

この資料での対応ルール
三菱UFJは、日銀が急変せず、10年国債利回りが緩やかに上昇し、与信費用が急増していない時に追加候補とします。日経平均が75日線を明確に下回る、与信費用見通しが悪化、または日銀後に円高が3%以上進む場合は追加を延期します。 銘柄へ戻る
自社株買い
自社株買いは、会社が市場から自分の株を買い戻す株主還元です。発行株数が減ると、同じ利益でも1株あたり利益、つまりEPSが上がりやすくなります。三菱商事は、利益の安定性と還元姿勢が評価ポイントです。

1. EPSを押し上げる効果
純利益が同じでも、発行株数が減れば1株あたり利益は増えます。市場はEPS成長を重視するため、自社株買いは株価の下支えになりやすいです。

2. 需給面の支え
会社自身が買い手になるため、下落局面で需給の支えになります。ただし、買付期間や上限金額が決まっているため、永続的な支えではありません。

3. 本業が弱い時の自社株買いは注意
資源価格下落や投資損失で利益が落ちている時に、自社株買いだけで株価を支えている場合は危険です。還元は大切ですが、本業のキャッシュフローが弱いと継続できません。

4. 商社特有のリスク
三菱商事は資源、非資源、事業投資の分散が強みですが、資源価格、為替、投資先評価損、海外景気の影響を受けます。原料炭、天然ガス、金属資源の市況が悪化すると利益見通しが下がります。

見る数字
自社株買い上限額、取得進捗率、総還元性向、EPS、営業キャッシュフロー、資源価格、通期利益見通しを見ます。

この資料での対応ルール
三菱商事は、通期利益見通しが維持され、自社株買いの進捗が確認でき、配当方針が維持される場合に追加候補とします。通期利益見通し-10%以上、資源価格急落、または自社株買い終了後に株価が75日線を下回る場合は追加停止。還元だけを理由に買い増しません。 銘柄へ戻る

守り枠
KDDIは、指数を大きく超える主役ではありません。役割は、配当、通信収入の安定性、下落局面での耐久力です。攻めの半導体や任天堂が揺れた時に、ポートフォリオ全体の値動きを落ち着かせるために入れます。

通信株は政府の料金政策や競争環境の影響を受けます。料金引き下げ圧力、楽天モバイルなど競争、通信障害、設備投資負担がリスクです。
