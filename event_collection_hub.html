上場・IPO・イベント収集コーナー

IPO、新規上場、適時開示、業績修正、自社株買い、株式分割、指数採用など、株価に影響しやすいイベントを集めて、銘柄候補の確認に使うための入口です。

更新: 2026/05/18 14:07

このコーナーの目的

IPOを拾う新規上場や大型IPOで資金がどこへ向かうかを見る
開示を拾う上方修正、下方修正、自社株買い、増配を確認する
危険を拾う減配、下方修正、上場廃止、監理銘柄を警戒する
スコアへ接続銘柄スコアの加点、減点、除外理由に使う
このページは、イベントを買い推奨に変えるものではありません。イベントを見つけた後、決算・PER・財務・株価反応で確認してから候補に入れます。

運用フロー

1. 収集JPX、TDnet、EDINET、Nasdaq、SECなどからイベントを拾う
2. 分類IPO、業績修正、還元、分割、M&A、指数採用に分ける
3. 重要度株価影響が大きいイベントだけ残す
4. 銘柄接続対象銘柄や関連セクターへつなぐ
5. 記録スコア変化と実際の株価反応を保存する

取得元と現実性

イベント取得元URL取得可能性取得方法信頼性制限モデルでの使い方
日本IPO・新規上場 JPX New Listings https://www.jpx.co.jp/english/listing/stocks/new/index.html 公開ページで確認可能 HTML/PDF/Excel等の公開情報を確認。まずはリンク収集と手動確認、次に自動抽出を検証。 高 ページ形式が変わる可能性あり。上場承認後の更新差分は都度確認が必要。 IPO、上場承認、上場日、事業内容、想定時価総額をイベント候補として記録する。
適時開示・決算・修正・自社株買い JPX TDnet / Company Announcements https://www.jpx.co.jp/english/listing/disclosure/index.html 公開ページで直近情報を確認可能 会社コード、開示日時、表題、PDFリンクを収集。重要語句で分類する。 高 無料表示は期間制限あり。長期履歴や高度検索は有料DBや個別取得が必要。 下方修正、上方修正、増配、自社株買い、株式分割、M&A、業績予想変更をスコア補正に使う。
有価証券報告書・大量保有・正式提出書類 EDINET API https://disclosure2.edinet-fsa.go.jp/ API利用可能 EDINET APIで書類一覧と提出書類を取得。企業財務・大株主・リスク情報の根拠に使う。 高 APIキー、書類種別、XBRL解析が必要。即時ニュース収集より実装工数が大きい。 海外売上比率、事業リスク、大株主、財務諸表の確認に使う。
米国IPO Nasdaq IPO Calendar https://www.nasdaq.com/market-activity/ipos 公開ページで確認可能 IPO予定日、ティッカー、会社名、取引所、想定価格帯を確認。 中〜高 Nasdaq側も予定日は推定扱いが含まれる。SEC提出書類で再確認が必要。 AI、宇宙、防衛、半導体など大型IPOが既存銘柄や指数に与える影響を見る。
米国正式開示 SEC EDGAR https://www.sec.gov/search-filings 公開検索/APIあり S-1、10-K、10-Q、8-Kを確認。米国IPOや大型開示の裏取りに使う。 高 日本語資料ではない。企業名やCIKの紐付けが必要。 米国IPO、AI企業、米国競合、上場準備企業の正式情報を確認する。
指数採用・除外 JPX / Nasdaq / S&P Dow Jones等 https://www.jpx.co.jp/english/markets/indices/ 公開ページで確認可能なものが多い 指数構成変更、採用候補、除外候補をイベントとして記録。 中〜高 指数ごとに発表元が異なる。全指数を一括で無料取得するのは難しい。 指数採用による需給イベント、除外による売り圧力を確認する。

分類ルール

分類見る言葉モデル上の扱い注意点
IPO・新規上場新規上場、IPO、上場承認、listing approval関連セクターの資金流入候補として記録上場直後は値動きが荒い。NISA本体候補ではなくテーマ確認に使う
業績修正上方修正、下方修正、guidance、forecast revision上方は加点候補、下方は除外または監視へ表題だけで判断せず通期予想、営業利益、EPSを見る
自社株買い自己株式取得、自社株買い、share repurchase株主還元点に加点候補規模、期間、発行済株式比率を確認
増配・減配増配、減配、dividend revision還元点と守り評価へ反映一時的な記念配当と継続増配を分ける
株式分割株式分割、stock split流動性改善イベントとして記録分割自体は企業価値を増やさない
M&A・提携買収、TOB、業務提携、capital alliance事業テーマ適合とリスクを確認買収価格、資金負担、希薄化を確認
指数採用指数採用、index inclusion、additions需給イベントとして記録発表後に織り込み済みの場合あり
上場廃止・監理上場廃止、監理銘柄、delisting原則除外・警戒NISA候補から外す方向で確認

項目意味入力例使い方

next_check次に確認すること決算短信PDF、会社予想、出来高反応人間が迷わないようにする
