# NISA日本株モデルポートフォリオ

顧客説明用のHTMLサイトです。

- `index.html`: NISA 1年保有テスト向けの入口(候補10社・240万円/口座)
- `development_plan.html`: 投資分析システム開発計画の1枚説明シート
- `development_plan.pdf`: 開発計画シートのPDF版
- `prototype_sample.html`: データ取得から点数出力までの暫定プロトタイプ

内容:

- S&P500投信・日経平均/TOPIXを**12か月で+1%ポイント以上**上回ることを目標として明示(短期3か月判定は915の劣後ルールで別管理)
- **1口座240万円**(NISA成長投資枠上限)の運用モデル
- 段階投入: 6/18初回35%・7/15第2回20%・8/17第3回15%・現金待機30%
- 銘柄別の数値根拠
- 2026年5月から8月の投資カレンダー
- 条件分岐と銘柄別アクションルール
- `data/market_update.json`を読み込む最新市場データ欄(index.html)
- GitHub Actionsによる平日15:40ごろの市場データ自動更新
- DXY、ドル円、米10年金利、金、VIXを使ったドル急落リスク監視

実売買前には、証券会社画面と最新の決算資料で数値を再確認してください。
