# NISA日本株モデルポートフォリオ

顧客説明用のHTMLサイトです。

- `index.html`: NISA日本株モデルポートフォリオの顧客向けサイト
- `development_plan.html`: 投資分析システム開発計画の1枚説明シート
- `development_plan.pdf`: 開発計画シートのPDF版
- `prototype_sample.html`: データ取得から点数出力までの暫定プロトタイプ

内容:

- S&P500投信・日経平均/TOPIXを3か月で+1%ポイント以上上回ることを目標として明示
- 100万円のNISA運用モデル
- 25万円、35万円、25万円、現金15万円の段階投入プラン
- 銘柄別の数値根拠
- 2026年5月から8月の投資カレンダー
- 条件分岐と銘柄別アクションルール
- `data/market_update.json`を読み込む最新データ欄
- GitHub Actionsによる平日15:40ごろの市場データ自動更新
- DXY、ドル円、米10年金利、金、VIXを使ったドル急落リスク監視

実売買前には、証券会社画面と最新の決算資料で数値を再確認してください。
