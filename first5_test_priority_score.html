3. 計算式

項目重み計算内容目的
公式決算スコア35%売上前年比25% + 利益前年比35% + EPS前年比25% + 直近2決算の改善度15%公式決算で本当に伸びているかを見る
市場反応スコア25%60日騰落率30% + 1年騰落率25% + 200日線乖離25% + 出来高比率20%市場が数字を評価しているかを見る
財務・割高リスク20%ROE45% + PER30% + PBR15% + 配当利回り10%成長に対して高すぎないかを見る
下落耐性15%60日最大下落率55% + 200日線過熱20% + 1年過熱15% + 出来高過熱10%NISA 1年保有で途中の大きな揺れに耐えられるかを見る
データ信頼度5%公式資料確認を基本80点、ローカル自動抽出成功率で最大20点加点今後の自動運用に載せやすいかを見る
特別ペナルティ減点利益マイナス、極端な上昇、極端な乖離、大幅下落、投資会社性などを減点強いが買いにくい銘柄を分ける

4. 現時点の読み方

  • アドバンテストは公式決算の強さが最も分かりやすい。高PER・高PBRのため、次は決算後の株価反応と受注見通しを確認する。
  • 古河電工は利益改善が強いが、1年上昇率と200日線乖離が極端。分析対象として重要だが、買い急ぎは危険。
  • フジクラは成長は強いが、60日最大下落率が大きい。NISA 1年保有では途中下落耐性を重点確認する。
  • レーザーテックはテーマ性が強い一方、直近営業利益前年比がマイナス。半導体テーマだから買う、という判定はしない。
  • ソフトバンクGは投資会社性が強く、他4社と同じ営業利益モデルで比較しない。別枠でNAVや投資損益を見る。

5. 次工程

  • この5社のうち上位と注意銘柄を分け、10社テスト候補へ拡張する。
  • 決算発表日後の株価反応を日次データと結合し、決算が本当に株価に効いたかを確認する。
  • 6月の市場イベント前後で、同じスコアがどう動くかを記録する。
  • 購入検討は、スコアだけでなく6月の市場条件チェックを通した後に行う。
